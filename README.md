# slack-sales-copilot
IA para Slack que ajuda vendedores a analisar oportunidades de venda e sugerir o próximo passo no processo comercial.

Slack Sales Coach é um bot de IA que auxilia vendedores durante o processo de vendas.

A partir do contexto de uma oportunidade (empresa, estágio do funil, objeções, concorrentes), o bot analisa o cenário e sugere:

- sinais de risco no negócio 
- estratégia de abordagem
- próximo passo recomendado no processo de vendas
